Exercise 3 Visual multicollinearity check (correlation matrix)

  • Build a correlation matrix among your regressors (X's).
  • Display it as a color-coded heatmap: values near -1 are toward one end of the scale; values near +1 are toward the other.
  • Also inspect the numeric values and flag any pair of regressors with absolute correlation > 0.8 as a potential multicollinearity issue.

Why it matters (summary): Multicollinearity can inflate the variance of the OLS estimator. As a result, coefficients may appear statistically insignificant even if there is a real effect when predictors are not so collinear.

Report: List any problematic regressor pairs you find (or write that there is no evident multicollinearity).

from itertools import combinations

INPUT_PATH = "/Users/eshanmomin/Desktop/MAIN/Emory_Junior_2025/ECON320/Week 7/usa_00004_cleaned.csv"

# Load dataset
df = pd.read_csv(INPUT_PATH)

# Define regressors to check
regressors = [
    "HHINCOME", "AGE", "EDUC"
]

# Keep only columns that exist in your data
regressors = [r for r in regressors if r in df.columns]
corr = df[regressors].corr(numeric_only=True)

# ============================================
# ======== PRINT AND FLAG HIGH CORR ==========
# ============================================
print("\n Correlation Matrix:\n")
print(corr.round(2))

# Flag pairs with |r| > 0.8
print("\n  Potential multicollinearity (|r| > 0.8):")
problematic_pairs = []
for a, b in combinations(regressors, 2):
    r = corr.loc[a, b]
    if abs(r) > 0.8:
        print(f" - {a} and {b}: {r:.2f}")
        problematic_pairs.append((a, b, r))

if not problematic_pairs:
    print(" No evident multicollinearity found.")

# ============================================
# ============ HEATMAP VISUAL ================
# ============================================
plt.figure(figsize=(10, 8))
sns.heatmap(
    corr,
    annot=True,
    fmt=".2f",
    cmap="coolwarm",
    center=0,
    linewidths=0.5
)
plt.title("Correlation Matrix among Regressors")
plt.tight_layout()
plt.show()

import pandas as pd
import statsmodels.api as sm
import statsmodels.formula.api as smf

# =====================================================
# ================= LOAD DATA =========================
# =====================================================
df = pd.read_csv("/Users/eshanmomin/Desktop/MAIN/Emory_Junior_2025/ECON320/Week 7/usa_00004_cleaned.csv")

# Keep only needed columns (drop missing)
cols = ["VALUEH", "HHINCOME", "AGE", "EDUC"]
df = df[[c for c in cols if c in df.columns]].dropna()

# =====================================================
# ============== RUN OLS REGRESSION ===================
# =====================================================

# Using formula notation (easier and cleaner)
model = smf.ols("VALUEH ~ HHINCOME + AGE + EDUC", data=df).fit()

# =====================================================
# ============== DISPLAY RESULTS ======================
# =====================================================
print(model.summary())

# Optional: export regression table to CSV
results_df = pd.DataFrame({
    "Variable": model.params.index,
    "Coefficient": model.params.values,
    "Std. Error": model.bse.values,
    "t-Statistic": model.tvalues.values,
    "p-Value": model.pvalues.values
})
results_df.to_csv("/Users/eshanmomin/Downloads/regression_results.csv", index=False)
print("\n Regression table saved to regression_results.csv")

Key Coefficients

Household Income (HHINCOME): The coefficient of 1.73 means that, holding other factors constant, each additional $1 of household income is associated with about $1.73 higher reported home value. The positive and highly significant effect (p < 0.001) supports the idea that wealthier households own higher-value homes.

Age of Householder (AGE): Each extra year of age corresponds to roughly $270 higher home value, controlling for income and education. This small but positive effect may reflect accumulated wealth or ownership of older, paid-off properties.

Education Level (EDUC): For each higher education category (e.g., moving from high-school some college bachelors), home value rises by about $7,975. This strong, significant relationship suggests that more-educated householders occupy more expensive homes, consistent with higher lifetime earnings and financial literacy.

Alignment with Hypothesis These results align closely with the earlier hypothesis that higher income and education are associated with higher home values, even after accounting for age. All key predictors have the expected positive signs and are statistically significant.

Surprises or Concerns

The R-squared = 0.15 indicates that the model explains about 15 percent of variation in home valuesreasonable for socioeconomic data but suggesting that structural (e.g., rooms, year built) and geographic factors still play large roles.

The warning about a large condition number implies possible numerical scaling or mild multicollinearity (likely between income and education), though not severe enough to distort signs.

Residual diagnostics (Durbin-Watson 0.69, high skew/kurtosis) show non-normal residuals, common with large-scale income and housing data, hinting that log-transforming skewed variables could improve model fit.
